# Portfolio Website – Lalith Santhosh C R

A modern, responsive personal portfolio website built using HTML and CSS. This project showcases my skills, projects, and contact information through a clean, professional dark-themed design.

---

## Features

- Responsive layout for desktop and mobile
- Fixed navigation bar
- Hero section with profile image and introduction
- Projects showcase section
- Contact section with social links
- Lightweight and fast (no frameworks)

---

## Tech Stack

- HTML5
- CSS3
